sendFeedback(action, lockToken, config, done)

This method sends the feedback action to the IoT Hub.

Parameters:
Name Type Description
action String

This parameter must be equal to one of the following possible values:

Value Action
abandon Directs the IoT Hub to re-enqueue a notification message so it may be received again later.
reject Directs the IoT Hub to delete a notification message from the queue and record that it was rejected.
complete Directs the IoT Hub to delete a notification message from the queue and record that it was accepted.
lockToken String

An HTTP E-Tag used to manage concurrent updates to the device
config Object

This is a dictionary containing the following keys and values:
